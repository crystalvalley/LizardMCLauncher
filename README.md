# 🦎 Lizard MC Launcher
도마뱀 컴퍼니 마인크래프트 서버 전용  모드팩 런처입니다.

---

## 📦 프로젝트 구조

| 프로젝트 | 설명 |
|----------|------|
| 🖥️ **Launcher** | WPF 기반 데스크톱 런처 클라이언트 |
| 🌐 **ServerSide** | 버전 관리, 공지사항, 리소스 배포를 위한 API 서버 |

## ✨ 주요 기능

- 🔐 Microsoft 계정 로그인
- ⚙️ NeoForge 자동 설치 및 모드 파일 동기화
- 🔄 런처 자체 업데이트
- 📢 공지사항 표시
- 🎮 게임 설정 (메모리, JVM 인수, 해상도, 서버 자동 접속)

---

## 🛠️ 기술 스택

| 구분 | 기술 |
|------|------|
| 🖥️ 클라이언트 | WPF (.NET 10), CmlLib, RestSharp |
| 🌐 서버 | ASP.NET Core (.NET 10), Entity Framework Core, PostgreSQL |
